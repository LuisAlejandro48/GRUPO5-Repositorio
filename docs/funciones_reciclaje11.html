Funciones del proyecto Reciclaje11

Este documento explica, una por una, las funciones que forman la app Clasificador de Reciclaje. La idea es leerlo como una guía del código, sin palabras raras ni relleno.

MainActivity.java

onCreate(Bundle savedInstanceState)

Prepara la pantalla principal. Conecta los botones de cámara, galería e historial con sus acciones.

revisarPermisoCamara()

Comprueba si la app ya tiene permiso de cámara. Si lo tiene, abre la cámara. Si no, pide el permiso.

abrirCamara()

Abre la pantalla de cámara en tiempo real.

abrirResultadoConUri(Uri uri)

Abre la pantalla de resultado con la imagen que el usuario eligió desde la galería.

mostrarAvisoPermiso()

Muestra un aviso cuando el permiso de cámara no fue concedido. Desde ahí se puede ir a Ajustes.

CameraActivity.java

onCreate(Bundle s)

Inicia la pantalla de cámara, prepara ML Kit, carga el modelo TensorFlow Lite y configura el botón para tomar foto.

iniciarCamara()

Configura CameraX. Activa la vista previa, la captura de foto y el análisis de frames en vivo.

analizarFrame(ImageProxy proxy)

Recibe cada frame de la cámara. Evita analizar dos frames al mismo tiempo, manda la imagen a ML Kit y cierra el frame al terminar.

mostrarResultadoEnVivo(List<DetectedObject> objects, ImageProxy proxy, int imageWidth, int imageHeight)

Convierte el frame a bitmap, clasifica el material con el modelo, toma el objeto más grande detectado y actualiza el cuadro y los textos en pantalla.

objetoMasGrande(List<DetectedObject> objects)

Busca el objeto con el rectángulo más grande. Se usa para dibujar el cuadro principal.

areaDe(Rect rect)

Calcula el área de un rectángulo multiplicando ancho por alto.

nombreDelObjeto(DetectedObject object)

Toma la etiqueta que ML Kit le dio al objeto y la traduce a un texto más claro, como Comida u Objeto del hogar.

tomarFoto()

Captura una foto, la guarda temporalmente y abre la pantalla de resultado para analizarla con más calma.

jpegABitmap(ImageProxy image)

Convierte una foto capturada en formato JPEG a un Bitmap listo para mostrarse o guardarse.

yuvABitmap(ImageProxy image)

Convierte un frame de cámara en formato YUV a Bitmap. Ese Bitmap es el que recibe el modelo para clasificar.

convertirANv21(ImageProxy image)

Reordena los planos Y, U y V del frame en formato NV21 para poder convertirlo a imagen JPEG.

rotar(Bitmap bitmap, int degrees)

Corrige la orientación de la imagen según la rotación reportada por la cámara.

mostrarEstado(String title, String detail, String instruction)

Actualiza los textos de la cámara en el hilo principal.

mostrarToast(String msg)

Muestra un mensaje corto en pantalla.

onDestroy()

Cierra el detector, el clasificador y el executor cuando la pantalla deja de usarse.

ResultActivity.java

onCreate(Bundle savedInstanceState)

Prepara la pantalla de resultado, carga el modelo y empieza el análisis de la imagen recibida.

cargarImagen()

Lee la imagen desde la galería o desde una foto tomada con la cámara.

detectarObjeto(Bitmap image)

Manda la imagen a ML Kit para encontrar objetos y sus etiquetas generales.

mostrarResultado(Bitmap image, List<DetectedObject> objects)

Clasifica el material, dibuja el cuadro si hay objeto, muestra la confianza, el objeto detectado y la instrucción de reciclaje.

objetoMasGrande(List<DetectedObject> objects)

Escoge el objeto más grande de la imagen.

areaDe(Rect rect)

Devuelve el área del rectángulo de un objeto.

guardarResultado()

Guarda la imagen y el resultado en la base de datos del historial.

mostrarMensaje(String message)

Muestra un estado cuando no se puede cargar la imagen, analizarla o abrir el modelo.

nombreDelObjeto(DetectedObject object)

Lee la mejor etiqueta de ML Kit y la convierte en un texto simple. Si no hay etiqueta, devuelve No identificado.

onDestroy()

Libera recursos cuando la pantalla se cierra.

TfliteClassifier.java

TfliteClassifier(Context context)

Carga el modelo TFLite, carga las etiquetas y lee el tamaño de entrada y salida del modelo.

clasificar(Bitmap bitmap, Rect box)

Prepara la imagen, ejecuta el modelo y devuelve el material con su confianza.

buscarEtiquetaMetal()

Busca la posición de la clase Metal en las etiquetas. Ayuda cuando el modelo duda con objetos metálicos.

recortar(Bitmap bitmap, Rect box)

Recorta la imagen usando el cuadro del objeto. Si no hay cuadro válido, usa la imagen completa.

cargarModelo(Context context)

Lee el archivo model_unquant.tflite desde assets y lo pone en un ByteBuffer.

cargarEtiquetas(Context context)

Lee labels.txt y limpia los números iniciales de cada etiqueta.

close()

Cierra el intérprete de TensorFlow Lite.

Result(String label, float confidence)

Guarda el resultado de una clasificación: etiqueta y confianza.

RecyclingClassifier.java

RecyclingClassifier()

Constructor privado para evitar crear objetos de esta clase. Solo contiene funciones estáticas.

normalizarCategoria(String label)

Convierte etiquetas parecidas en una categoría limpia: Plástico, Vidrio, Papel, Cartón, Metal o Basura.

instruccionPara(String category)

Devuelve el contenedor recomendado para la categoría.

colorPara(String category)

Devuelve el color usado para pintar el cuadro del objeto.

fondoPara(String category)

Devuelve el color de fondo del mensaje de instrucción.

limpiarTexto(String value)

Pasa texto a minúsculas y quita acentos para comparar sin errores.

OverlayView.java

Box(Rect rect, String label, int color)

Guarda la información necesaria para dibujar un cuadro: posición, texto y color.

OverlayView(Context context, AttributeSet attrs)

Prepara los pinceles que se usan para dibujar cuadros y textos.

mostrarCajas(List<Box> newBoxes, int imageWidth, int imageHeight)

Recibe las cajas nuevas y fuerza que la vista se vuelva a dibujar.

onDraw(Canvas canvas)

Dibuja los rectángulos sobre la imagen, ajustando la escala al tamaño de la pantalla.

HistorialActivity.java

onCreate(Bundle savedInstanceState)

Prepara la pantalla de historial, la barra superior, la lista y la base de datos.

prepararLista()

Configura el RecyclerView y su adaptador.

cargarHistorial()

Lee los registros guardados y cuenta cuántos hay por categoría.

actualizarResumen(int plastico, int vidrio, int papel, int carton, int metal, int basura)

Muestra el resumen de categorías guardadas.

mostrarDialogoEliminar(ObjetoReciclado item)

Pregunta antes de borrar un registro.

eliminarRegistro(ObjetoReciclado item)

Borra la imagen guardada, elimina el registro de Room y recarga la lista.

onDestroy()

Apaga el executor de base de datos al cerrar la pantalla.

HistorialAdapter.java

alMantenerPresionado(ObjetoReciclado item)

Función de la interfaz que se ejecuta cuando el usuario mantiene presionado un registro.

HistorialAdapter(List<ObjetoReciclado> items, OnItemLongClickListener longClickListener)

Recibe los datos iniciales y la acción para pulsación larga.

actualizarDatos(List<ObjetoReciclado> newItems)

Reemplaza la lista del historial y refresca la vista.

onCreateViewHolder(ViewGroup parent, int viewType)

Crea la vista de cada fila del historial.

onBindViewHolder(ViewHolder holder, int position)

Llena una fila con categoría, etiqueta, confianza, fecha e imagen.

getItemCount()

Devuelve cuántos registros hay en la lista.

ViewHolder(ItemHistorialBinding binding)

Guarda las referencias de una fila del historial.

AppDatabase.java

objetoDao()

Da acceso a las operaciones de la tabla de objetos reciclados.

obtenerBaseDeDatos(Context context)

Crea o devuelve la única instancia de la base de datos Room.

ObjetoDao.java

insertar(ObjetoReciclado objeto)

Guarda un registro nuevo en el historial.

obtenerTodos()

Devuelve todos los registros, empezando por el más reciente.

eliminar(ObjetoReciclado objeto)

Elimina un registro del historial.

contarPorCategoria(String cat)

Cuenta cuántos registros hay de una categoría específica.

ObjetoReciclado.java

ObjetoReciclado(String categoria, float confianza, String etiquetaImageLabeling, long fechaEscaneo, String rutaImagen)

Crea un registro con categoría, confianza, etiqueta del objeto, fecha e imagen guardada.
